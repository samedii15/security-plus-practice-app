# Security Policy

## Reporting Security Vulnerabilities

If you discover a security vulnerability within CyberAcademy, please send an email describing the issue and reproduction steps. All security vulnerabilities will be promptly addressed.

Please include the following information in your report:
- Description of the vulnerability
- Steps to reproduce the issue
- Potential impact
- Any suggested fixes (if available)

## Scope

This security policy applies to:
- The main application code
- Database interactions
- Authentication and authorization mechanisms
- API endpoints

Thank you for helping keep CyberAcademy secure!
