# Screenshots

This directory will contain screenshots of the application for the README.

## Recommended Screenshots

1. **Landing Page** - Main login/registration screen
2. **Exam Interface** - Active exam with questions and timer
3. **PBQ Screen** - Performance-based question examples
4. **Analytics Dashboard** - User performance and analytics
5. **Admin Panel** - Admin feedback and user management

## Adding Screenshots

1. Take high-quality screenshots (1280x720 or higher)
2. Save as PNG format with descriptive names:
   - `landing-page.png`
   - `exam-interface.png`
   - `pbq-example.png`
   - `analytics-dashboard.png`
   - `admin-panel.png`
3. Add them to the main README.md with proper markdown image syntax
